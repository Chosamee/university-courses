PROFILE

ヨルシカ

2017년 4월 21일, 구두의 불꽃으로 활동을 시작한 2인조 밴드.

2019년 뮤직 비디오들의 누적 유튜브 조회수가 1억 회를 돌파하고 구독자 수가 100만을 돌파하며 J-POP의 신흥 강자로 떠올랐다.

밴드명 '요루시카'는 구름과 유령의 '이제 밤밖에 잠들지 못하고' 라는 가사의 '夜しか(밤밖에, 요루시카)'에서 따온 것이다.


멤버

n-buna[ナブナ] （Guitar / Composer） : 보카로P 출신으로 작사, 작곡, 편곡을 담당하고 있으며 밴드의 주축이다.

suis[スイ] （Vocal） : 보컬을 담당하고 있다. 특유의 니힐한 목소리에서 나오는 독특한 음색이 특징이다.
